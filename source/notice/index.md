# 技术架构

使用 [Hexo](https://hexo.io/) + Github Pages 部署，Cloudflare ~~减速云~~ 负责加速（至少比 Github 快），Github repo 在 [https://github.com/worker2011/worker2011.github.io/](https://github.com/worker2011/worker2011.github.io/)，可以去里面的 Discussion 更改评论。主题是 ~~[Next](https://theme-next.js.org/)~~[Redefine](https://redefine-docs.ohevan.com/)，全站使用 Google 的免费 SSL 证书，开启 https。域名的提供商是 [Eu.org](https://nic.eu.org)，也是免费的。所以，这个站点是小L用纯免费方法搭建的（逆天……）

可以到[https://github.com/worker2011/worker2011.github.io/commits/main/](https://github.com/worker2011/worker2011.github.io/commits/main/)查看所有的更新commit记录，2024年8月24日起用上了[GPG Keys](https://spencerwoo.com/blog/wait-this-is-not-my-commit)进行身份验证，**此后**没有标明verified的都不是小L的commit！
